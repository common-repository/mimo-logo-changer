This plugin can customize wordpress Login page easily.

== Description ==

This plugin created to change the Logo, Link, Title of wordpress login page. also this plugin can hide navbar icon in wordpress dashboard.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mimo-logo-changer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Wordpress Logo Changer to configure the plugin
